Algorithms

Conjugate Gradients

There are four different conjugate gradient implementations, all with exchangeable termination criterion. The default criterion is based on the relative energy error.

Available implementations:

  • Conjugate Gradients (CG)
  • Truncated Conjugate Gradients (TCG)
  • Regularized Conjugate Gradients (RCG)
  • Truncated Regularized Conjugate Gradients (TRCG)

Newton Methods

There is a generic Newton method that allows to customize termination criterion and damping strategy.

Available implementations:

  • a local Newton method
  • an affine covariant Newton method
  • an affine contravariant Newton method

Composite Step Methods

There is a composite step method for nonconvex optimization (the normal solver can be customized).

Available Implementations:

  • an affine covariant composite method

Vector Spaces

Available Implementations:

  • a real space
  • product spaces that can be nested to group variables
  • finite dimensional vector spaces based on Eigen
  • function spaces based on FEniCS and/or Kaskade7

Adapters

Currently there are adapters for

  • Eigen (for finite dimensional problems)
  • FEniCS
  • Kaskade7
