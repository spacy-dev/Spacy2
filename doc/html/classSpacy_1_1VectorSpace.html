Detailed Description

Function space \((X,\|\cdot\|)\).

To construct a function space you need to provide a VectorCreatorAnchor "VectorCreator". It allows to customize the type of vector that is created by the vector space. Possibly, it provides additional information.

See also
VectorCreator.
